## ADDED Requirements

### Requirement: 設定探索優先順位の一貫性
システムは設定ファイル探索において、公開されている優先順位を一貫して適用しなければならない。

#### Scenario: XDG環境変数がある場合に既定XDGパスより優先される
- **GIVEN** `XDG_CONFIG_HOME` が設定されている
- **AND** プロジェクト設定ファイルが存在しない
- **WHEN** システムがグローバル設定ファイルの候補を解決する
- **THEN** `XDG_CONFIG_HOME/cflx/config.jsonc` が優先候補として扱われる
- **AND** CLI公開挙動は変化しない

#### Scenario: 非推奨経路を整理しても設定読込結果は維持される
- **GIVEN** 既存ユーザーが従来どおり設定ファイルを配置している
- **WHEN** 設定を読み込む
- **THEN** 読み込まれる設定内容と優先順位は変更されない
