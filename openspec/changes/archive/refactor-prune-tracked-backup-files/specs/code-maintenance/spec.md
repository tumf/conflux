## ADDED Requirements

### Requirement: バックアップ拡張子ファイルの追跡禁止
リポジトリは `.bak`、`.backup`、`.bak2` 拡張子のバックアップファイルを追跡してはならない。

#### Scenario: 追跡ファイル検査でバックアップが検知されない
- **GIVEN** リポジトリの追跡ファイル一覧を取得する
- **WHEN** バックアップ拡張子（`.bak` / `.backup` / `.bak2`）で照合する
- **THEN** 一件も検知されない

#### Scenario: 新規混入が自動検知される
- **GIVEN** バックアップ拡張子ファイルが新規追加される
- **WHEN** プロジェクトの検証手順を実行する
- **THEN** 検証は失敗し、混入が通知される
