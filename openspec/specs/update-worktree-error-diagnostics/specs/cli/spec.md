## ADDED Requirements
### Requirement: Log Panel Shows Full Message Content

TUI のログパネルは、ログメッセージの全文を保持し、表示幅を超える場合は折り返して表示しなければならない（MUST）。

折り返し表示によってもログのスクロール挙動は維持されなければならない（MUST）。

#### Scenario: 長文ログは折り返して表示される
- **GIVEN** ログパネルに表示幅を超える長文メッセージが追加される
- **WHEN** TUI がログを描画する
- **THEN** メッセージは折り返され、全文が画面内に表示される

#### Scenario: 折り返し表示でもスクロールが維持される
- **GIVEN** 折り返しが発生する長文ログが複数存在する
- **WHEN** ユーザーがログパネルをスクロールする
- **THEN** 既存のスクロール操作（Page Up/Down、マウスホイール）が引き続き機能する
